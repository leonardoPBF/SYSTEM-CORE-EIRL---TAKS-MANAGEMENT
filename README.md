# TaskSystemCore EIRL - Sistema de Gestión

Sistema completo de gestión desarrollado para TaskSystemCore EIRL.

## Estructura del Proyecto

```
.
├── frontend/          # Aplicación React + Vite + TypeScript
└── backend/           # API Express + TypeScript
```

## Instalación

### Frontend

```bash
cd frontend
npm install
```

### Backend

```bash
cd backend
npm install
cp .env.example .env
```

## Desarrollo

### Frontend

```bash
cd frontend
npm run dev
```

El frontend estará disponible en `http://localhost:3000`

### Backend

```bash
cd backend
npm run dev
```

El backend estará disponible en `http://localhost:5000`

## Tecnologías

### Frontend
- React 18
- Vite
- TypeScript
- React Router

### Backend
- Express
- TypeScript
- CORS
- dotenv
